# Roadmap

## Phase 1: MVP Foundation

- Monorepo structure (`apps/web`, `tools/scraper`, docs, workflows)
- Angular onboarding + results pages
- Champion cards with progressive disclosure
- Static JSON data loading
- Scheduled scrape + Pages deployment automation

## Phase 2: Data Reliability

- Validate and tune `window=current` source mapping
- Add retries / lane-level scrape diagnostics
- Add schema checks in CI for produced JSON
- Track scrape freshness and failure statuses in manifest

## Phase 3: UX Improvement

- Better sort presets by role and playstyle
- Optional advanced filters (ban-rate thresholds, min games)
- Saved filters in URL + local storage
- Better mobile density and card interactions

## Phase 4: Product Expansion

- Add more regions
- Add matchup-context panels
- Add patch history comparisons
- Add i18n and copy variations (ES/EN)
